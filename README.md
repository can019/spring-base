# Spring-base
## description
- 개인 프로젝트에 사용 될 Spring-boot boiler plate입니다.
- 현재 template-repostiroy로는 설정하지 않았습니다.

## Docs
[Github page](https://can019.github.io/spring-base)

## Docker 설치
- mac x86에서만 확인되었습니다.

`./init.sh` 실행

``` shell
./init.sh
```
## 환경
- Java version: 21
  - distribution: 'temuri
- Gradle version: 8.7
